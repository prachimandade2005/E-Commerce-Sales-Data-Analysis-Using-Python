# 📊 E-Commerce Sales Data Analysis Using Python

## 📌 Project Overview
This project analyzes an **E-Commerce sales dataset** using Python to understand sales trends, customer behavior, and business performance. The goal of this project is to extract meaningful insights from the data that can help businesses improve decision-making and overall performance.

The project includes **data cleaning, exploratory data analysis (EDA), and data visualization** using Python libraries.

---

## 🎯 Objectives
- Analyze sales performance across different product categories
- Understand the impact of **discounts on sales**
- Identify **product return patterns**
- Analyze **delivery time vs customer ratings**
- Study **payment method behavior and return rates**

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset Description
The dataset contains the following information:

- Order ID
- Product Category
- Price
- Discount
- Payment Method
- Delivery Time
- Customer Rating
- Return Status

---

## ⚙️ Project Workflow

### 1. Data Loading
- Import dataset using **Pandas**
- View dataset structure and basic information

### 2. Data Cleaning
- Check and handle **missing values**
- Remove **duplicate records**
- Ensure proper **data types**

### 3. Exploratory Data Analysis (EDA)
- Sales analysis by product category
- Discount impact on order volume
- Delivery time impact on customer ratings
- Return rate analysis
- Payment method comparison

### 4. Data Visualization
Various graphs were created to understand patterns such as:
- Category-wise revenue
- Discount vs order count
- Return rate analysis
- Payment method comparison
- Delivery time vs customer rating

---

## 📊 Key Insights

- Electronics category generated the **highest revenue**
- Fashion category had **high order volume but lower revenue**
- Faster delivery resulted in **better customer ratings**
- Electronics had a **higher return rate**
- Moderate discounts (10–20%) improved sales performance
- Cash on Delivery showed **higher return rates**

---

## 📈 Conclusion
The project demonstrates how **data analysis can help e-commerce businesses understand customer behavior, optimize pricing strategies, and improve operational efficiency**.

Using Python-based analysis, meaningful insights can be extracted to support **data-driven decision making**.
